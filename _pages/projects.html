Asteroids game that uses the boids algorithm to simulate flocking. A k-d tree is used to determine the nearest neighbors. (JavaScript, HTML5 Canvas)

Classic snake game with a current score and a high score from the current browser session. (JavaScript and JQuery)

pollfor.us

Live audience polling web application. User creates a poll, other users vote their answer with results displayed in realtime. Used Twilio API to send HTTP requests to server from phone and used Pusher API to push results back in realtime. Built using Ruby On Rails, Backbone.js, JavaScript, JQuery, MongoDB, and care.
